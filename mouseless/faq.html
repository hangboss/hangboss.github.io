FAQs

Can't open the software in macOS?
Open the Terminal and type the following command: xattr -d com.apple.quarantine /Applications/Mouseless.app
codesign --force --deep --sign - /Applications/Mouseless.app If after doing this, the software still cannot be opened, please try the following operations: Settings - Security & Privacy - Accessibility - Allow Mouseless. If you have already performed this operation, please remove Mouseless and perform it again.
